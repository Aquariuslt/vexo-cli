An Vue version Aexo Blog App


## Installation

```bash
npm install -g vexo
```

## Usage

### Initial 
```bash
mkdir blog
cd blog
vexo init 
```

### Serve
```bash
vexo serve
```

### Deploy
```bash
vexo deploy
```
